Colors

The colors of the tiles are based on 3 factors:
  • Course The first three characters of the course code determine the course. All courses in the same category will have similar hues.
  • Course Type Specifically, the fifth and sixth character of the course describe the type of course it is. Different course types will differ in hue slightly.
  • Grade The fourth character determines the grade (or level) of the course. The higher the grade, the darker the cell will be.

Dots

The last two digits of your course code is your class code. The number of dots equals the value of the class code.

If there is a dash instead of dots, the user has chosen to not enter their class code.
